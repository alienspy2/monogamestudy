enum RectpackSharp.PackingHints

Specifies hints that help optimize the rectangle packing algorithm.

열거형 멤버
TryByArea 

Tells the rectangle packer to try inserting the rectangles ordered by area.

TryByPerimeter 

Tells the rectangle packer to try inserting the rectangles ordered by perimeter.

TryByBiggerSide 

Tells the rectangle packer to try inserting the rectangles ordered by bigger side.

TryByWidth 

Tells the rectangle packer to try inserting the rectangles ordered by width.

TryByHeight 

Tells the rectangle packer to try inserting the rectangles ordered by height.

TryByPathologicalMultiplier 

Tells the rectangle packer to try inserting the rectangles ordered by a pathological multiplier.

FindBest 

Specifies to try all the possible hints, as to find the best packing configuration.

UnusualSizes 

Specifies hints to optimize for rectangles who have one side much bigger than the other.

MostlySquared 

Specifies hints to optimize for rectangles whose sides are relatively similar.
